🔍 ¿Qué es “There is / There are”?

Se usa para decir que algo existe o está presente en un lugar.

📋 Estructura básica:

FormaSingular / IncontablePlural
✅ AfirmativaThere is a cat / some milkThere are two books
❌ NegativaThere isn’t a chair / any waterThere aren’t apples
❓ InterrogativaIs there a problem?Are there any students?

📌 Cuándo usar cada uno:

There is → There is a dog / There is some water
There are → There are many people

❗Palabras clave comunes:

There is some juice / There aren’t any bananas
Is there much time? / Are there many students?

🔄 Variaciones útiles:

Tiempo verbalEjemplo
FuturoThere will be a test tomorrow
Presente perfectoThere has been a mistake
PasadoThere was a problem / There were books

✍️ Práctica rápida:

A. Completa:
___ a spider in the bathroom → ✅ There is
___ many tourists → ✅ There are

B. Forma negativa:
There is a chair → ❌ There isn’t a chair
There are apples → ❌ There aren’t apples

C. Preguntas:
There is a problem → ❓ Is there a problem?
There are cookies → ❓ Are there cookies?

🧠 Mini resumen:

  • ✅ There is → singular o incontable
  • ✅ There are → plural
  • ❌ Negativa → isn’t / aren’t
  • ❓ Pregunta → Is there...? / Are there...?
